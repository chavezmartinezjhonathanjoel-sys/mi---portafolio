## 🚀 Cómo Usar

### 1. **Abrir en el navegador**
   - Simplemente abre `index.html` en tu navegador favorito

### 2. **Personalizar contenido**
   
   **En `index.html`:**
   - Reemplaza "Tu Nombre" con tu nombre real
   - Actualiza los enlaces de navegación
   - Personaliza cada sección con tu información
   - Agrega tus proyectos reales
   - Actualiza tus habilidades técnicas
   - Configura tus datos de contacto
   - Agrega enlaces a redes sociales

### 3. **Personalizar diseño**
   
   **En `styles.css`, cambia las variables de color:**
   ```css
   :root {
       --primary: #6366f1;        /* Color principal */
       --secondary: #ec4899;      /* Color secundario */
       --dark: #1f2937;           /* Texto oscuro */
   }
   ```

### 4. **Agregar imágenes**
   - Crea una carpeta `images/` en el mismo nivel que `index.html`
   - Reemplaza los placeholders con tus imágenes
   - Actualiza las rutas en `index.html`

## 🎨 Colores Disponibles

La plantilla usa un sistema de colores que puedes personalizar:

- **Primario:** `#6366f1` (Indigo)
- **Secundario:** `#ec4899` (Rosa)
- **Oscuro:** `#1f2937`
- **Claro:** `#f9fafb`

Todos los colores están definidos en las variables CSS, así que cambiarlos es muy fácil.

## 📊 Ejemplo de Personalización

### Cambiar colores principales:
```css
:root {
    --primary: #3b82f6;      /* Azul */
    --secondary: #10b981;    /* Verde */
    --dark: #1f2937;
}
```

### Agregar un nuevo proyecto:
```html
<div class="project-card">
    <div class="project-image">
        <img src="images/proyecto.jpg" alt="Mi proyecto">
        <div class="project-overlay">
            <a href="https://tuproyecto.com" class="btn-icon"><i class="fas fa-external-link-alt"></i></a>
        </div>
    </div>
    <div class="project-info">
        <h3>Mi Proyecto</h3>
        <p>Descripción del proyecto</p>
        <div class="project-tags">
            <span class="tag">Tech1</span>
            <span class="tag">Tech2</span>
        </div>
    </div>
</div>
```

## 🚀 Despliegue

### GitHub Pages (Gratis)
1. Sube los archivos a un repositorio
2. Ve a Settings > Pages
3. Selecciona la rama main
4. ¡Listo! Tu sitio estará en `https://usuario.github.io/repo`

### Netlify (Gratis)
1. Conecta tu repositorio de Git
2. Autoriza a Netlify
3. ¡Automático! Se deploya cada que hagas push

### Vercel (Gratis)
1. Conecta tu repositorio
2. Confirma la configuración
3. Tu sitio estará online en segundos
